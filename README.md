# Unofficial Parker-Damon Information Hub

A parent-maintained, unofficial information resource for the Parker-Damon school community.

## Publishing

This site is designed for GitHub Pages using Jekyll.

1. Upload all files in this repository.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`.
5. Save.

The site should publish at:

`https://eamacm.github.io/unofficial_parker_damon_info_hub/`

## Updating the site

- Add dated updates in `_posts/`
- Edit FAQs in `_data/faq.yml`
- Edit board roles in `_data/board.yml`
- Edit official links in `_data/links.yml`
- Replace the Google Form URL in `questions.md`
- Replace the Google Calendar URL in `calendar.md`
